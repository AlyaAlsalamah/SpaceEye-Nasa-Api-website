About

Our distributed systems project introduced us to the fascinating world of loosely coupled components communicating via APIs. Through this project, we successfully developed our first distributed system, which employed a direct request-response communication style between a client (web browser) and a server. Our primary objective was to design a captivating website that showcases NASA's daily picture, leveraging an API key for access [1][3][4]. To aid in the implementation of the JavaScript portion, we found valuable guidance in a video resource [2].
